## 4. 준수 체크리스트

### 4.1 출시 전 필수

- [ ] 개인정보 처리방침 작성 및 게시
- [ ] 이용약관 법률 검토 완료
- [ ] 개인정보 영향평가 (해당 시)
- [ ] 보안 취약점 점검 완료
- [ ] 암호화 적용 확인 (저장 + 전송)
- [ ] 접근 로그 보관 정책 적용

### 4.2 운영 중 정기

| 주기 | 점검 항목 |
|-----|----------|
| 월간 | 접근 권한 리뷰 |
| 분기 | 취약점 점검 |
| 반기 | 개인정보 처리 현황 점검 |
| 연간 | 침투 테스트, 인증 갱신 |
